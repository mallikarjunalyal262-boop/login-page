# Login Page Project

This project is a simple login page developed using HTML and CSS.  
It is designed to demonstrate the basic structure of a web page and the use of CSS for styling.

## Description
The login page includes input fields for username and password along with a login button.  
CSS is used to improve the layout, alignment, and overall appearance of the page.

## Technologies Used
- HTML
- CSS

## Project Files
- login.html
- style.css

## How to Run the Project
Open the login.html file in any modern web browser such as Google Chrome, Microsoft Edge, or Firefox.

## Purpose of the Project
This project is created for learning and academic purposes to understand basic web development concepts.
